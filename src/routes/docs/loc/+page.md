:::info[LOC Calculation]
The popular [cloc](https://github.com/AlDanial/cloc) tool is used to count the lines of code in the `src/lib` folder of the GitHub repository. Each version entry specifies the tool version, the command line used, and any other relevant data.

The total count is also shown without `RouterTrace`, which is a non-production component. The lines of code for the `RouterTrace` component are also counted using `cloc` with the path `.\src\lib\RouterTrace`.

For detailed information about changes between releases, see the projectâ€™s [Releases page](https://github.com/WJSoftware/svelte-router/releases).
:::
